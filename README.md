# Bizunehe — Professional Websites for Small Businesses

I help small businesses build clean, modern websites that attract customers and build trust online.

Features
- Business website design (1–5 pages)
- Mobile-responsive layouts
- Contact & WhatsApp integration
- Basic SEO setup
- Website updates & support

Quick start
1. Clone: `git clone https://github.com/bizunehejifarabedada-create/bizunehe-website.git`
2. Open `index.html` in your browser or serve with a static server (e.g. `npx serve .`)
3. Deploy to GitHub Pages, Vercel, or Netlify.
